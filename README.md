# Spotter

## About
Spotter is a game that lets you practise recognising birds.

## Technologies
- Nx monorepo
- Next.js
- Node.js

## Usage

1. Clone the repository
```
git clone https://github.com/marissaverdonck/birdapp.git
```
2. Install 
```
yarn
```
3. Start app
```
yarn start:next-app
yarn start:api
```
